%% Outer PID: Altitude → Velocity command
err_alt = alt_ref(k) - alt(k);
int_alt = max(min(int_alt + err_alt * dt, 5e3), -5e3);   % anti-windup
d_alt   = (err_alt - err_alt_prev) / dt;
vel_cmd = 2.80*err_alt + 0.35*int_alt + 1.10*d_alt;
vel_cmd = max(min(vel_cmd, 50), -50);                       % saturate

%% Inner PID: Velocity → Thrust
err_vel = vel_cmd - vel(k);
int_vel = max(min(int_vel + err_vel * dt, 200), -200);
thrust  = m * (5.50*err_vel + 0.80*int_vel + 0.40*d_vel);
thrust  = max(min(thrust, 1.5), -1.5);                      % thruster limit [N]

// 05 — Capabilities

System Features

🎛️
Cascaded PID
Outer altitude loop feeds inner velocity loop. Anti-windup clamping prevents integrator saturation during large setpoint changes.
🔌
HIL Simulation
Full Hardware-in-the-Loop validated at 50+ Hz via UDP/Serial bridge between MATLAB plant and embedded MCU.
📡
SDR + NRF Telemetry
GFSK-modulated 433 MHz SDR downlink + NRF24L01+ 2.4 GHz uplink. 40+ signal exchanges per cycle with <2% packet loss.
🛸
3-Axis Attitude
Independent PID controllers for Roll, Pitch, and Yaw with reaction wheel torque limits and derivative-on-measurement.
🌍
Orbital Disturbances
Models J2 oblateness, LEO atmospheric drag, solar radiation pressure, and stochastic sensor noise.
📊
Control Analysis
Automated Bode plots, Nyquist diagrams, pole-zero maps, and Kp/Kd sensitivity surface generation.
